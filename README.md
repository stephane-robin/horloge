Linux users: After downloading, right-click the file, go to Properties > Permissions, and check the box "Allow executing file as program". 
Then, simply double-click the file to start your clock!
Ready to go!

Utilisateurs Linux : Après le téléchargement, faites un clic droit sur le fichier, allez dans Propriétés > Permissions, et cochez la case "Autoriser l'exécution du fichier comme un programme".
Ensuite, double-cliquez simplement sur le fichier pour démarrer votre horloge !
Prêt à l'emploi !
